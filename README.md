# OverExpansion — Bank Rate Comparison Site

## Project structure

```
overexpansion/
├── index.html                        ← the website
├── scraper.py                        ← daily rate scraper
├── rates.json                        ← auto-generated by scraper (committed daily)
└── .github/
    └── workflows/
        └── scrape-rates.yml          ← GitHub Actions cron job
```

## How it works

1. **GitHub Actions** runs `scraper.py` every day at 8am UTC
2. `scraper.py` fetches:
   - **FDIC API** → official national average savings/CD rates
   - **Each bank's public rate page** → live APY for ~20 online banks
3. Results are saved to `rates.json` and committed back to the repo
4. `index.html` fetches `rates.json` on page load and shows live rates
5. If `rates.json` is missing or stale, the site falls back to baseline rates

## Setup (takes ~5 minutes)

### 1. Create a GitHub repo
- Go to github.com → New repository → name it `overexpansion`
- Upload all files from this folder

### 2. Enable Actions write permissions
- Go to repo Settings → Actions → General
- Under "Workflow permissions" → select **Read and write permissions**
- Click Save

### 3. Deploy the site (pick one)

**Option A — GitHub Pages (free)**
- Settings → Pages → Source: Deploy from branch → select `main` → folder `/`
- Your site will be live at `https://yourusername.github.io/overexpansion/`

**Option B — Netlify (free, custom domain)**
- netlify.com → New site → Import from GitHub → select your repo
- Build command: (leave blank) · Publish directory: `.`

### 4. Trigger first scrape manually
- Go to Actions tab in your repo
- Click "Daily Rate Scraper" → "Run workflow" → Run
- This writes the first `rates.json` immediately

### 5. Verify
- Open your site URL
- The banner should show "Live rates — last scraped [today]"
- APY cells will show a green "● live" indicator

## Troubleshooting

**"Rates were last scraped X days ago"**
→ Check Actions tab for failed runs. Some banks may have updated their page structure. Fix the regex in `scraper.py` for that bank.

**A specific bank shows "baseline" instead of "● live"**
→ That bank's page blocked the scraper or changed its HTML. Open `scraper.py`, find the scraper function for that bank, and update the URL or regex.

**FDIC API returns no national average**
→ The FDIC summary API endpoint may have changed fields. Check `banks.data.fdic.gov/docs` for updated field names.
